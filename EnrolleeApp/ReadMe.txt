Для запуска программы дважды кликните на файл EnrolleeApp.exe

Для функционирования приложения понадобится база данных MySQL. Вы можете её создать при помощи приложения Oracle MySQL Workbench. Скрипт для создания базы данные находится в папке CREATE DATABASE.

Тестовы данные для приложения находятся в папке TEST.
